# Andjintrans LLC

A modern, bilingual corporate website for Andjintrans LLC — a Mongolian freight forwarding and logistics company specializing in dangerous goods transportation.

## Tech Stack

- **Next.js 16** - React framework with App Router
- **TypeScript** - Type-safe JavaScript
- **Tailwind CSS 4** - Utility-first CSS framework
- **Framer Motion** - Animation library
- **next-intl** - Internationalization (Mongolian & English)
- **Vercel** - Deployment and hosting

## Features

- Bilingual support (Mongolian / English) with seamless locale switching
- Dark hero section with full-bleed background image and gradient overlay
- Scroll-aware header with dark-to-light transition on homepage
- Staggered entrance animations and scroll-triggered reveals
- Mobile-first responsive design with slide-out navigation
- Animated gradient background on inner pages

## Pages

- **Home** - Hero with company overview, DG specialization badge, and key statistics
- **About** - Company history, mission, and team information
- **Services** - Freight forwarding services:
  - Sea Freight
  - Air Freight
  - Rail Freight
  - Road Transport
  - Transit Shipping
  - Customs Clearance
  - Cargo Insurance
  - Domestic Transport
  - Export Services
- **Resources** - Industry knowledge base:
  - Incoterms
  - Container Types
  - Wagon Types
  - Packaging Guidelines
  - Dangerous Goods Classification
  - Shipping Documents
- **News** - Company updates and industry news
- **Contact** - Quote request form

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.
